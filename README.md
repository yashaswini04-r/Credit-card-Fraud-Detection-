This project uses machine learning to detect fraudulent credit card transactions by analyzing historical data. It aims to protect customers and banks from financial losses by identifying suspicious transactions accurately.

Clone the repository
Install required libraries with pip install -r requirements.txt (e.g., scikit-learn, pandas, numpy)
Download the dataset 
Set up the environment as described in the instructions file
